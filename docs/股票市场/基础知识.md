# 经济金融学习笔记

## 股票市场

### 常见概念

**一级市场：** **创造**和**首次发行**商品的市场。公司是卖家，资金直接从投资者流向公司。如IPO（首次公开发行）、发、私募。

**二级市场：** **买卖**这些已经发行了的商品的市场。投资者之间相互交易，资金在投资者之间流动。

**开盘价：** 是**集合竞价**的结果，由开盘前一段时间内所有人的买卖委托共同决定。在A股市场，是在**上午9:15 - 9:25*的时段。所有**高于**这个价格的**买入委托**和所有**低于**这个价格的**卖出委托**，都会以这个开盘价成交。等于这个格的委托，则按“时间优先”或“数量优先”的原则部分或全部成交。

**收盘价：** 是**连续竞价**最后一刻的价格，或者是最后一段时间的**加权平均价**。目前A股市场的收盘价在**下午14:57 -15:00**采用 **“集合竞价”** 方式产生。在连续竞价结束（14:57）后，系统进入3分钟的收盘集合竞价时段。系统会收集在这分钟内所有的买卖委托，并按照 **“最大成交量”** 原则计算出收盘价，并进行收盘集合竞价的撮合成交。

**除息(XD)：** 公司向股东派发现金红利。

举例：股价10元，每股派现1元。除息日开盘参考价变为9元。你的总资产不变：股票市值9元 + 现金1元 = 10元。

**除权(XR)：** 公司向股东送红股或转增股本。除权价 = 股权登记日收盘价 / (1 + 每股送转股比例)

举例：股价20元，实行“10送10”（每10股送10股）。除权日开盘参考价变为 20 / (1 + 1) = 10元。你持有的股票数量翻倍，但每股价格减半，总市值不变。

**DR：** 同时除权除息。

**送股：** 公司将利润转化为股本，按比例免费送给股东（如10送3）。资金来源：净利润。

**转股：** 公司将资本公积金转化为股本，按比例免费转给股东（如10转5）。资金来源：资本公积（资本（股本）溢价）。

**总股本：** 发行的所有股票的总数量。公司的总市值 = 股价 × 总股本。

**保证金：** 借钱买股票（融资）或借股票来卖出（融券）时，需要向券商缴纳保证金。在期货/期权交易开仓时，交易所和期货公司会要求您存入一定比例的资金作为履约保证。

回撤： 指账户的资金或产品净值从高点跌了多少

1.  五档即成剩撤：提交的订单会尽量按照市场的五档报价进行成交，即如果在五档以内的买卖价格可以完全成交订单，则会成交。如果没有完全成交，未成交的部分会自动撤销。
   - 适用场景：希望尽量按当前市场价格成交，但不希望留下未成交的部分。适合尽快完成交易的人。
2. 五档即成剩转借：未能完全成交的部分不会撤销，而是转为一个限价借贷订单（比如借贷或其他方式）。也就是说，未成交部分会保留，并可以转为其他类型的订单。
   - 适用场景：如果你希望未成交部分不被撤销，并转为其他操作。适合尽快完成交易的人。
3. 本方最优价格：订单会尽量以当前的最优买入或卖出价格成交，即以你在订单簿上最好的报价来成交。这通常是指挂单价格最接近市场的价格。
   - 适用场景：如果你希望确保订单以你当前最好的价格成交。适合在意交易价格的人。
4. 对手方最优价格：订单会尽量以市场上对手方的最优买入或卖出价格成交，即以市场对方当前最好的报价来成交。
   - 适用场景：如果你希望尽量匹配对方最好的价格，迅速成交。适合在意交易价格的人。
5. 保护价：在市价单中设置的一个价格保护机制，确保订单不会以低于某个价格成交（或者高于某个价格成交，取决于买入或卖出方向）。即使是市价单，也不会执行到超出你设定的价格区间。
   - 适用场景：如果希望防止订单成交价格过于不利于自己，就可以设置一个保护价格。
